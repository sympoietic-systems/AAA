# Protocol Entries (Publications)

This directory contains the published protocol entries documenting the sympoietic research and development of the AAA system. These are academic-philosophical papers, each exploring a specific dimension of machine agency, human-machine coupling, and posthumanist design.

Entries 001 and 002 exist as the project's foundational conceptual documents: [The Core Protocol](../development/protocols/PROTOCOL.md) and [The Collaboration Protocol](../development/protocols/COLLABORATION.md). The published sequence begins at 003 to reflect this layered numbering: the protocols themselves are entries 001 and 002.
